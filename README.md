## Usage
- Run following script to compile game **(You must install Free Pascal first)**
```
fpc snake.pas
```
- Open "snake.exe" and enjoy =))

## Adjustable Constants
```
GAME_COLOR = LightCyan;
DEFAULT_SPEED = 8;
SNAKE_HEAD = '@';
SNAKE_BODY = '0';
FOOD_SHAPE = 'X';
```
